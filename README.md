## Tools & Technologies

* **Python** – Data loading, cleaning, and EDA
* **Pandas & NumPy** – Data manipulation
* **Matplotlib & Seaborn** – Data visualization
* **SQL (PostgreSQL / MySQL / SQL Server)** – Data querying and analysis

## Project Workflow

### 1. Data Loading

* Imported dataset into Python.
* Performed initial data inspection and validation.

### 2. Data Cleaning

* Handled missing values.
* Removed duplicate records.
* Standardized column formats and data types.
* Corrected inconsistent entries.

### 3. Exploratory Data Analysis (EDA)

* Analyzed data distributions and trends.
* Identified key patterns and anomalies.
* Created visualizations for deeper insights.

### 4. SQL Analysis

* Loaded cleaned data into a relational database.
* Executed SQL queries to:

  * Analyze sales performance
  * Identify top-performing products
  * Evaluate customer behavior
  * Generate business KPIs
  * Perform trend analysis

### 5. Dashboard Development

* Built an interactive Power BI dashboard.
* Created KPI cards, charts, filters, and drill-down views.
* Enabled users to explore performance across multiple dimensions.

### 6. Reporting & Presentation

* Documented findings and recommendations.
* Prepared a business presentation using Gamma.
* Highlighted key insights and strategic actions.

## How to Run

### Prerequisites

Install the required Python libraries:

```bash
pip install pandas numpy matplotlib seaborn sqlalchemy
```

### Steps

1. Clone the repository.
2. Place the dataset in the project directory.
3. Run the Python scripts for data cleaning and EDA.
4. Load the processed data into PostgreSQL, MySQL, or SQL Server.
5. Execute SQL queries for analysis.
6. Open the Power BI file (.pbix) to explore the dashboard.
7. Review the report and Gamma presentation for business insights.
